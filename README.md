# SurveyScriber Web

Survey data extraction tool using AI and OCR.

## Setup

### Backend
`ash
cd backend
pip install -r ../requirements.txt
python ../backend_entry.py
`

### Frontend
`ash
cd frontend
npm install
npm run dev
`

Open http://localhost:5173

## Features
- AI-powered data extraction (OpenAI, Anthropic, Google Gemini)
- Multiple OCR options (Google Vision, Azure, Custom API)
- Excel/CSV export
